# Bio Heroes 《生物英雄传》

## 项目概述
一款龙珠Z风格的生物科学卡牌对战网页游戏。父子亲子项目。
让小朋友在对战的快感中自然吸收生物学知识——玩法本身就是在学，而不是"学完了奖励你玩"。

核心参考：Dragon Ball Card Warriors（龙珠Z卡卡罗特内置卡牌游戏）。

## 技术栈

### 前端
- **React 18 + Vite** — 核心框架
- **Tailwind CSS** — 样式
- **Framer Motion** — UI 动画：卡牌翻转、抽卡动画、页面转场、手牌展开/收拢
- **WAAPI (Web Animations API)** — 战斗演出：攻击序列、命中闪光、震屏效果（已替代 GSAP）
- **tsParticles** — 粒子特效：爆炸碎片、技能光效、环境事件氛围

### 后端（后期接入）
- **Supabase** — Auth + PostgreSQL + Realtime（预留多人对战）

### 部署
- **Vercel**

### 动画分工
```
Framer Motion（UI 层）→ 页面转场、卡牌翻转、手牌展开、能量条、按钮反馈
WAAPI（战斗演出层）  → 攻击序列、受伤抖动、觉醒演出、进化爆发演出
tsParticles（粒子层）→ 技能特效、爆炸碎片、环境事件粒子、能量聚集光效
```

### 性能策略
- 低端设备降级：减少粒子、简化动画
- 提供"精简模式"开关

## 核心设计原则

1. **好玩优先**：先是好玩的游戏，其次才是教育工具
2. **科学准确**：卡牌数值、技能、描述基于真实科学事实
3. **龙珠精神**：收集成就感、组队策略深度、战斗觉醒爽感
4. **玩法即学习**：知识融入核心机制，不是答题奖励
5. **允许失败**：答错也有部分奖励
6. **亲子友好**：7岁能理解，大人也有策略深度
7. **中文为主**：科学名词附带英文

## 卡牌系统

### 卡牌类型

| 类型 | 说明 |
|------|------|
| 生物卡 | 主力战斗卡，有 ATK / HP 两项基础属性 + 技能 |
| SP觉醒卡 | 条件触发后从 SP 卡组翻出，逆转战局 |
| 环境事件卡 | 战斗中自动触发，影响场上所有生物 |

### 卡牌属性

每张生物卡包含：
- **卡名**（完整中文名，如"蚂蚁军团·觅食狂潮"）
- **阵营**：自然系 nature 🌱 / 人体系 body 🧬 / 病原系 pathogen 🦠 / 科技系 tech ⚗️
- **能量费用 cost**：0-10（与稀有度独立）
- **稀有度 rarity**：R / SR / SSR（与费用独立）
- **ATK**：攻击力
- **HP**：生命值（归零被击败）
- **技能**：基于真实科学属性（技能列表见下方）
- **科学知识卡**：一段有趣的科学事实
- **进化链**（部分卡牌）

> **注意：没有 DEF 和 SPD 基础属性。** 防御和速度相关能力全部通过技能实现，与 Card Warriors 一致。

## ⚠️ 费用与稀有度：两个独立维度

**这是本游戏最重要的设计规则之一。**

参考 Dragon Ball Card Warriors：费用和稀有度完全独立。

### 费用（Cost 0-10）= 什么时候能上场

- 费用由卡牌的战场即时价值决定（ATK + HP + 技能强度）
- 低费卡早期上场，高费卡后期上场
- 高费卡属性总值更高，但"每费效率"递减

### 稀有度（R / SR / SSR）= 抽到的难度 + 技能独特性

- R：容易抽到，技能简单或无技能
- SR：较难抽到，技能有特殊机制
- SSR：极难抽到，技能独特且可改变战局

### 关键：同费用内可以有不同稀有度

```
蚂蚁军团           → cost 0, R    — 无技能，纯炮灰
蚂蚁军团·觅食狂潮   → cost 2, SR   — 技能"我们一起出发"全队ATK+20%
蚂蚁军团·蚁后之令   → cost 3, SSR  — 技能"倾巢而出"召唤额外蚂蚁

T细胞              → cost 1, SR   — 基础免疫技能
T细胞·觉醒         → cost 3, SSR  — 技能"精准打击"忽略守护

水熊虫·不灭        → cost 2, SSR  — ATK低但免伤技能极强（低费高稀有度典型）
```

## 卡牌技能系统

所有高级战斗能力通过技能实现，不是基础属性。参考 Card Warriors 的 ability 系统：

### 通用技能（跨系可用）

| 技能 | 效果 | 科学包装举例 |
|------|------|-------------|
| **守护 (Guard)** | 该卡在场时，对手只能攻击该卡 | 骨骼铠甲 — 骨骼保护内脏 |
| **迅击 (Swift Attack)** | 打出当回合即可攻击（无召唤疲劳） | 神经元闪电 — 神经信号传导极快 |
| **穿透 (Piercing Strike)** | 攻击守护卡时，溢出伤害穿透到主人 | T细胞·觉醒 — 精准穿透细胞膜 |
| **压制 (Overpower)** | 击杀对方卡后，溢出伤害转给主人 | 蓝鲸巨灵 — 体型碾压 |
| **突进 (Rush)** | 直攻主人时伤害翻倍 | 金鹰猎手 — 俯冲猎杀 |
| **自愈 (Natural Recovery)** | 每回合自动回复一定 HP | 水熊虫 — 隐生状态自我修复 |

### 专属技能（特定卡牌独有）

通过卡牌设计工具创建，基于真实科学属性设计。例如：
- 白血球战士 — "免疫风暴"：对病原系卡牌伤害 +50%
- 红血球运输者 — "氧气补给"：全队回复 HP
- 章鱼智者 — "墨汁迷雾"：本回合对方攻击有 30% 概率 miss
- 心脏引擎 — "脉动加速"：全队本回合攻击时额外造成 20% 伤害

### 技能设计原则
- 技能越强 → 基础属性（ATK/HP）越低
- 同费用同稀有度内，有技能的卡 ATK/HP 要低于无技能的卡
- 技能必须有科学依据，不能凭空编造

## 数值平衡系统（Power Curve）

### 费用-属性对应原则

> 以下为参考范围，具体数值在卡牌设计工具中校验。
> ATK/HP 的单位与 Card Warriors 对齐（千位数），主人 HP 为 30000。

```
cost 0：  ATK+HP 总和 ≤ 2000，通常无技能
cost 1：  ATK+HP 总和 ≤ 4000
cost 2：  ATK+HP 总和 ≤ 6000
cost 3：  ATK+HP 总和 ≤ 8000
cost 4-5：ATK+HP 总和 ≤ 12000（后续扩卡）
cost 6+： ATK+HP 总和 12000+（终极卡，需要 bank 释放，后续扩卡）
```

### 技能估值扣减

有技能的卡在同费用档内，ATK+HP 总和要低于无技能的卡：
```
简单技能（单一效果）：扣减约 10-15% 属性
中等技能（条件触发/双重效果）：扣减约 15-25%
强力技能（改变战局）：扣减约 25-40%
```

### 伤害计算
```
攻击对方卡牌：双方互扣
  → 我方 ATK 扣对方 HP，对方 ATK 同时扣我方 HP
  → HP 归零的卡被击败

直攻主人：
  → 我方 ATK 直接扣主人 HP（主人不反击）

觉醒加成（答对问答后）：
  → 答对：ATK ×2.0
  → 答错但接近：ATK ×1.3
  → 答错：ATK ×1.0（正常攻击）
```

### 数值反映真实科学
- 蓝鲸 HP 最高（地球最大动物）
- 神经元有迅击技能（神经信号传导最快）
- 水熊虫有免伤/自愈技能（极端环境存活能力）
- 心脏是支援型（为全身供血 → 全队增益）
- 蚂蚁单体弱但群体增益强（社会性昆虫）
- 骨骼有守护技能（保护内脏器官）

## 卡组与对战系统

### 卡库与卡组

**卡库（Collection）**：玩家通过抽卡获得的所有卡牌。

**出战卡组（Deck）**：

| 区域 | 数量 | 规则 |
|------|------|------|
| 主卡组 | 20 张生物卡 | 同一张卡（完整卡名）最多 3 张 |
| SP 卡组 | 3 张 SP 觉醒卡 | 同一张卡最多 1 张 |

### 卡组编辑器（Deck Builder）

- 展示卡库全部卡牌，按系列/稀有度/费用筛选
- 选入主卡组（20 张）和 SP 卡组（3 张）
- 实时显示：能量曲线（各费用段各几张）、阵营构成
- 卡组不满 20 张时无法进入对战

### 主人（Leader）系统

- 所有玩家的主人相同：**30000 HP**，纯被动，无技能，不上场
- **胜利条件**：将对方主人 HP 打至 0
- **守护机制**：场上有 Guard 技能的卡时，对手只能攻击守护卡；无守护卡时可直攻主人
- 后期可扩展：可选不同主人（各有专属被动技能）

### 对战流程

#### 开局
1. 双方从 20 张主卡组洗牌
2. 抽 5 张起手手牌
3. 1 次 mulligan：任意张放回卡组底部，重抽等量
4. 随机决定先手（先手第一回合不抽牌）

#### 每回合流程
```
1. 抽牌阶段
   从卡组顶部抽 1 张（先手第一回合跳过）

2. 能量阶段
   当回合可用能量 +1（回合1=1, 回合2=2…上限 10）

3. 主阶段（出牌）
   · 花费能量从手牌打出生物卡到战场（最多 5 个战场位）
   · 一回合可打出多张（只要能量够）
   · 战场位满时可替换已有卡（被替换的卡进弃牌堆）
   · ⚠️ 本回合打出的卡不能攻击（召唤疲劳）
   · 有"迅击"技能的卡例外，出场即可攻击

4. 战斗阶段（攻击）
   · 选择场上已有的卡（非本回合刚打出的）发起攻击
   · 对方有守护卡 → 只能攻击守护卡
   · 对方无守护卡 → 可攻击对方场上卡 或 直攻主人
   · 攻击卡牌：双方互扣（我ATK扣对方HP，对方ATK扣我HP）
   · 直攻主人：我ATK直接扣主人HP（主人不反击）
   · HP 归零 → 被击败进弃牌堆
   · 战斗中 25% 概率触发问答觉醒

5. 结束阶段
   · 未用完的能量存入能量储备（bank）— ⏳ 后期实现
   · 手牌超过 7 张上限时弃掉多余的牌
   · 回合交给对方
```

### 能量系统

#### 基础能量
- 每回合可用能量 = 当前回合数（上限 10）
- 打出卡牌消耗对应 cost 的能量

#### 能量储备 / 进化爆发（⏳ 后期实现）

参考 Card Warriors 的 Energy Bank：
- **自动存储**：每回合未用完的能量自动存入 bank
- **一次性释放（进化爆发）**：整局可释放 1 次，储备能量全加到当回合（可突破上限 10）
- **释放后 bank 破坏**：使用后不再存储能量
- **技能修复**：特定卡牌技能可重新激活 bank（极稀有的强力效果）

## SP觉醒卡系统

参考 Card Warriors 的 SP Deck。

### 规则
- 构建卡组时选 3 张 SP 卡（不占主卡组 20 张名额）
- 同一张 SP 卡最多 1 张
- 不能从手牌打出，必须满足触发条件

### 触发条件（满足任一）
- 连续答对 2 道问答题
- 己方主人 HP 降至 50% 以下
- 战斗进入第 8 回合

### 触发流程
从 3 张 SP 卡中**随机翻出 2 张，玩家选 1 张**上场。

### 示例 SP 卡
| SP卡名 | 触发效果 |
|--------|----------|
| 🧬 细胞分裂·无限增殖 | 复活一名已倒下的队友，恢复50%HP |
| 🦕 远古召唤·霸王龙 | 召唤霸王龙助攻3回合，每回合造成 ATK 3000 伤害 |
| ❤️ 生命脉动·全系统协同 | 全队恢复40%HP，ATK +20% 持续2回合 |

## 环境事件系统

每 3 回合自动触发一个随机环境事件。

| 事件 | 效果 | 学到的知识 |
|------|------|-----------|
| 🌡️ 全球变暖 | 自然系 ATK -20% | 气候变化 |
| 🦠 病毒爆发 | 无人体系卡的队伍 -500HP/回合×2 | 免疫系统 |
| 🌧️ 暴风雨 | 全场 ATK -10%，自然系不受影响 | 天气与生态 |
| 🔥 森林大火 | 自然系 -1500HP，病原系不受影响 | 自然灾害 |
| 🌸 春天来了 | 全体+500HP，全自然系额外+1000HP | 季节与生态 |
| 🏭 栖息地破坏 | HP最高的卡 ATK -25% | 人类活动 |
| 🤝 共生效应 | 不同阵营的卡两两回复 400HP | 共生关系 |
| 🧪 基因突变 | 随机一卡 ATK ×1.5，HP ×0.7 | 基因变异 |

## 抽卡经济系统

## 重要提醒

- **属性系统**：只有 ATK + HP，没有 DEF 和 SPD
- **费用 ≠ 稀有度**：两个独立维度，绝不混淆
- **同一张卡 = 完整卡名相同**，同名最多 3 张
- **召唤疲劳**：打出当回合不能攻击（迅击例外）
- **攻击互扣**：打对方卡时双方同时扣 HP
- **守护优先**：有 Guard 卡必须先打
- **技能越强 → ATK/HP 越低**
- 生成新卡必须附科学知识
- React (JSX) + Tailwind CSS
- 战斗动画用 WAAPI（不用 GSAP）
- 中文为主
- 低端设备提供精简模式
